# 基于element-ui的后台管理系统


##灵感来自

[PanJiaChen/vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)

[sendya/ant-design-pro-vue](https://github.com/sendya/ant-design-pro-vue)


## 功能特性

1.echart
2.骨架屏组件
3.卡片组件
4.富文本（封装tinymce5，不支持低版本浏览器）
5.导出excel
6.视频播放（封装vue-video-player）
7.类浏览器标签栏
8.权限指令
9.数字格式化指令


##安装

```shell
$ git clone https://github.com/ChuHingYee/elementui-admin-vue-boilerplate.git
$ npm install
```

##运行

###开发环境

```shell
$ npm run serve
```
###生产环境

```shell
$ npm run build
```
###lint检查并修复

```shell
$ npm run lint
```

###单元测试

（暂无）
```shell
$ npm run test:unit
```

###生成element-ui样式变量

路径：/src/styles/element-variables.scss
```shell
$ npm run theme:var
```

###生成element-ui主题文件

路径：/src/theme
```shell
$ npm run theme
```
